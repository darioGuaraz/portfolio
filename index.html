Encuesta de Satisfacción de Servicio +

Con el fin de mejorar la calidad del servicio del área de soporte IT, se desarrolló un formulario en línea adaptativo y responsivo (Mobile first). En este, el usuario podrá, entre otras cosas, calificar la calidad y el tiempo de resolución/respuesta del técnico involucrado. Los datos ingresados en dicho formulario se almacenan en una base de datos relacional (SQL) en un equipo local. Esta base de datos puede consultarse mediante PowerBI, lo que permite realizar un análisis cualitativo y cuantitativo más detallado gracias al feedback del propio usuario/cliente. Aplicando este sistema en el proceso de calidad, se podrá medir y mejorar de forma continua el desempeño del área de soporte IT.
